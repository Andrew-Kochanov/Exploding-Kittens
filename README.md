Цель кейса:

Создать десктоп приложение позволяющее администрировать процесс игры в рамках одной партии:


записывать и проверять корректность ходов

вычислять промежуточные результаты

поддерживать инвариантность

хранить историю игр

вычислять рейтинги и статистики по игрокам
